# AI独立开发者变现工具平台 MVP - 项目对齐文档摘要

> 版本：V1.1 正式版 | 生效日期：2026-04-20 | 适用范围：项目全员6人

## 一、项目核心定位

**一句话定位**：专为AI时代独立开发者打造的「风口资讯→变现方案→需求验证」一站式轻量化工具平台

### 核心约束
- 团队：6人（3脱产 + 3非脱产）
- 周期：21天（7个3天周期）
- 目标：交付可演示MVP，支撑AI产品岗面试

### 最终交付物
- 公网可访问单页应用（全链路闭环）
- 面试物料（PPT、脚本、问答库）
- 完整代码仓库 + 项目文档

## 二、三大核心模块

### 2.1 资讯列表
- 50条精选独立开发者变现向资讯
- 100%贴合独立开发者/一人公司创业者
- 必须自带变现属性

### 2.2 方案生成器（6模块固定结构）
1. 核心风口机会
2. 细分用户与精准痛点
3. VibeCoding 快速落地 MVP 步骤
4. 零成本引流渠道与玩法
5. 需求验证核心 SOP
6. 成本与收益预估

### 2.3 需求验证工具箱
- 小红书笔记生成器
- 落地页生成器

## 三、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | React 18 + React Router 6 | 单页应用 |
| 后端 | Node.js + Express | API代理、解决跨域 |
| 数据 | 本地JSON | 无数据库 |
| AI接口 | 豆包/OpenAI API | 统一调用规范 |
| 部署 | Vercel | 公网免费链接 |
| 协作 | Git | 分支开发 |

## 四、参数传递规范

| 链路 | 参数 | 方式 |
|------|------|------|
| 资讯→方案 | infoTitle、infoContent、infoTags | Router + localStorage |
| 方案→工具箱 | projectName、targetUser、corePain、drainageChannel、projectDesc | Router + localStorage |

## 六、开发周期（21天/7个3天周期）

| 周期 | 里程碑 | 验收标准 |
|------|--------|----------|
| 1 Day1-3 | 规则锁死+Demo通 | 需求冻结+Demo通 |
| 2 Day4-6 | 单模块开发完成 | 页面可独立运行 |
| 3 Day7-9 | 全链路跑通 | 主流程无断点 |
| 4 Day10-12 | BUG修复+代码冻结 | 无致命BUG |
| 5 Day13-15 | 部署上线+物料完 | 可公网演示 |
| 6 Day16-18 | 演示打磨+预案全 | 演示丝滑 |
| 7 Day19-21 | 最终验收+面试准备 | 全员就绪 |

## 七、协作铁律

- **需求冻结**：第1周期后禁止新增功能
- **单人单责**：一项任务一个负责人
- **异步同步**：无固定会议，3天一同步
- **非脱产尊重**：工作日不@，仅晚8点后沟通
